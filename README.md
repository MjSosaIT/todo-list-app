# To-Do List App

## Description
A simple To-Do List application built using Python and Tkinter. This app allows users to add, delete, edit, and save tasks, providing a straightforward way to manage daily activities.

## Features
- Add tasks to the list
- Delete selected tasks
- Edit existing tasks
- Save tasks to a file
- Load tasks from a file

## Installation
1. Ensure you have Python installed on your machine.
2. Install Tkinter if it's not already included (usually comes with Python).
3. Download or clone this repository.
4. Run the application with the command: `python todo_list.py`.

## Usage
Simply run the application and start adding tasks. Use the buttons to manage your tasks effectively.
